# HW1 实验报告模板

## 1. 实验任务

本次作业使用 `NumPy` 从零实现一个三层多层感知机分类器，在 `Fashion-MNIST` 数据集上完成 10 类服装图像分类。实验要求包括：

- 自主实现自动微分与反向传播
- 实现 `SGD`、学习率衰减、交叉熵损失和 `L2` 正则化
- 在验证集上保存最优模型
- 通过超参数搜索比较不同设置的效果
- 在测试集上输出准确率与混淆矩阵
- 可视化第一层权重并分析空间模式
- 进行错例分析

## 2. 数据集与预处理

`Fashion-MNIST` 包含 10 个类别的灰度服装图像，图像大小为 `28 x 28`。

- 训练集：60000 张
- 测试集：10000 张
- 验证集：从训练集中按比例划分

预处理步骤：

1. 将图像展平为 `784` 维向量
2. 将像素值归一化到 `[0, 1]`
3. 使用训练集均值与标准差做标准化

## 3. 模型设计

模型结构为三层 `MLP`：

`784 -> hidden_dim -> hidden_dim -> 10`

其中：

- 输入维度为 `784`
- 两层隐藏层维度可调
- 输出层维度为 `10`
- 激活函数支持 `ReLU`、`Sigmoid`、`Tanh`

### 3.1 自动微分实现

本实验实现了轻量级计算图系统，支持以下运算的梯度传播：

- 加法
- 减法
- 乘法
- 矩阵乘法
- 求和 / 求均值
- `ReLU`
- `Sigmoid`
- `Tanh`
- `log_softmax`
- 张量切片

交叉熵损失通过 `log_softmax + NLL` 的方式实现。

- `outputs/eval/misclassified_samples.png`

可分析的原因包括：

- 类别之间外形接近，例如 `Shirt` 与 `T-shirt/top`
- 图像中目标轮廓不清晰
- 某些服装样式兼具多个类别的视觉特征
- 仅使用全连接层，缺少卷积结构，对局部空间模式利用不如 `CNN`

## 10. 总结

本实验成功使用 `NumPy` 从零实现了支持自动微分的三层 `MLP` 分类器，并完成了训练、验证、测试、超参数搜索和可视化分析。实验表明，多层感知机可以在 `Fashion-MNIST` 上取得较好的分类性能，但对于形状相近的服装类别仍存在混淆。后续可以尝试：

- 使用更深的网络
- 引入更合适的初始化策略
- 调整学习率衰减与正则化强度
- 使用卷积神经网络进一步提升性能
